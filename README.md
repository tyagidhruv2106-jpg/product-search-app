# Product Search App

This is a beginner React project that allows users to search and filter products.

## Features

- Search products by name
- Filter products by category
- Real-time product filtering
- Clean card-based UI

## Technologies Used

- React
- JavaScript
- CSS

## How to Run

1. Clone the repository
2. Run:

npm install

3. Start the project:

npm start
